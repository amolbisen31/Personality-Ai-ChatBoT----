# Personality-Ai-ChatBoT
AI chatbot with multiple personalities using Groq API

## Features
- Multiple personalities (Osho, Bill Gates, Mark Zuckerberg)
- Real-time responses
- Clean ChatGPT-style UI
- Enter-to-send support
- Personality switch with instant chat reset

## Tech Stack
- Python
- Flask
- Groq API
- HTML, CSS, JavaScript

## How to Run
1. Clone the repo
2. Create virtual environment
3. Install requirements
4. Add GROQ_API_KEY in .env
5. Run app.py

## Note
.env file is not included for security reasons.
